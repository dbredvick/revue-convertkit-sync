An application that can sync your Revue & ConvertKit newsletter subscribers.

To get started, head over to the [Getting Started](/getting-started) guide.

## Why do I need this?

If you're looking for an affordable (free) way to sync your Revue subscribers with your ConverKit account, this is the way.

By running the code yourself, you're able to use free tiers on popular platforms like [Vercel](https://vercel.com), [Supabase](https://supabase.io), and [GitHub Actions](https://github.com/features/actions) to keep your subscribers in sync.

## How does it work?

- Revue's API
- ConvertKit's API
- GitHub Action cron jobs
- Supabase storage
- Next.js serverless functions
- Deployed to Vercel
